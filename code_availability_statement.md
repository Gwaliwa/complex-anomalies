# Code Availability Statement

A controlled reproducibility package for the Werner–PSO decomposition-guided magnetic anomaly inversion workflow is provided with this submission. The package uses real manuscript outputs, including anomaly-peak tables, Werner batch solutions, target-ranking summaries, Euler solution outputs, drillhole-validation summaries, PSO result metadata, and figure-generation scripts.

The package does not redistribute the full production application, dense ranking automation, SaaS backend, or proprietary deployment components. The exact profile arrays used in the application can be exported from the Streamlit session using the included `export_real_profile_from_streamlit_session.py` script.

Large public geoscience datasets are not redistributed; data sources are described in the manuscript and repository documentation.
